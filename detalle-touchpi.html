TouchPi Node

Portable Edge AI Hybrid Infrastructure Platform · RPi 4B + LattePanda

Detalle del proyecto TouchPi Node

TouchPi Node es una plataforma híbrida portátil diseñada para ejecutar cargas de Edge AI, procesamiento distribuido e interoperabilidad Windows / Linux simultánea. El sistema resuelve la necesidad de contar con un nodo portátil capaz de ejecutar tareas de IA, actuar como entorno de desarrollo, controlar sistemas externos y mantener estabilidad térmica bajo cargas prolongadas.

La plataforma integra dos módulos de cómputo independientes — Raspberry Pi 4B (Linux, Edge AI) y LattePanda (Windows, desarrollo y control) — en un diseño multicapa modular con sistema de refrigeración activo de doble turbina delta BFB1012, flujo gradualizado, cámaras internas de presión y redirección angular de flujo diseñadas para enfriamiento simultáneo de ambos módulos.

El sistema incluye integración ergonómica de controles GameSir X2, conectividad LAN de baja latencia, compatibilidad con aceleración eGPU externa y administración de recursos compartidos entre ambas plataformas.

Ingeniería térmica

Sistema de Refrigeración Activo

Diseñado completamente para enfriamiento simultáneo del módulo superior e inferior, con arquitectura de presión gradualizada y redirección angular de flujo.

Arquitectura de Flujo Térmico

01

Doble turbina Delta BFB1012 — generación de presión estática activa con tecnología dual fan blade en ambos módulos.

02

Flujo de aire gradualizado — velocidad y presión distribuidas en etapas para maximizar cobertura sin crear turbulencias parásitas.

03

Cámaras internas de presión — canales sellados que concentran el flujo sobre los componentes críticos de cada módulo.

04

Redirección angular de flujo — deflectores posicionados para dirigir el aire hacia hotspots específicos de RPi 4B y LattePanda.

05

Enfriamiento simultáneo — módulo superior e inferior reciben flujo activo de forma independiente sin interferencia entre capas.

Diferenciadores técnicos

Innovaciones Destacadas

Integración Híbrida Windows/Linux

Interoperabilidad simultánea entre módulo Linux (RPi 4B) y módulo Windows (LattePanda) con administración de recursos compartidos y comunicación entre plataformas.

Diseño Multicapa Modular Portátil

Arquitectura de módulos apilados independientes con acceso individual para mantenimiento, actualización o sustitución de componentes sin desmontar el sistema completo.

Sistema Térmico de Presión Gradualizada

Doble turbina BFB1012 con flujo gradualizado, cámaras internas de presión y redirección angular para enfriamiento simultáneo de ambos módulos — 45°C máx. en RPi 4B compilando.

Adaptación Ergonómica GameSir X2

Integración de controles GameSir X2 adaptados ergonómicamente al form factor del sistema para interacción física con la plataforma en escenarios de campo.

Integración LAN de Baja Latencia

Conectividad de red local de baja latencia integrada para comunicación entre módulos y con infraestructura externa, compatible con entornos NAS y sistemas distribuidos.

Compatibilidad eGPU Externa

Soporte para aceleración gráfica externa, permitiendo escalar el rendimiento de inferencia AI más allá de las capacidades integradas de cada módulo de forma portátil.
