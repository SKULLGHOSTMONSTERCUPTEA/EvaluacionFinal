# Arquitectura de Sistemas Adaptativos e Integración IA

**Portafolio técnico de Alexis Negrón Rodríguez** — diseño, optimización y validación empírica de sistemas operativos personalizados, ingeniería térmica experimental e inferencia de IA en hardware restringido.

🔗 **[Ver el portafolio en vivo](https://github.com/SKULLGHOSTMONSTERCUPTEA/EvaluacionFinal)**

---

## Sobre este portafolio

Este repositorio contiene un ecosistema de **8 proyectos** de R&D en hardware y software, cada uno validado empíricamente bajo carga sostenida — no benchmarks sintéticos. Cubre desde la optimización de Windows 11 a nivel de Kernel hasta el reacondicionamiento físico de GPUs de servidor, pasando por un sistema operativo personalizado (WinOs Unify Corps) con agentes de IA integrados.

La documentación técnica completa suma más de 120 páginas entre el documento extendido, la documentación física y los anexos de NexCore.

## Resultados destacados

| Proyecto | Resultado |
|---|---|
| Mainframe eGPU (Tesla V100) | Hasta **93 tokens/s** en inferencia local de Codestral 22B |
| Asus E210M (Win 11 LTSC) | **920 MB** de RAM en frío con Defender, Firewall y BitLocker activos |
| NexCore (Getac K120) | **42+ horas** de entrenamiento CPU-only sin degradación térmica |
| CryoStorm v2 | Eliminación de spikes térmicos de ~90°C mediante rediseño de flujo de aire |
| Asus E210M — red | Pico de **1,445 MB/s** en transmisión sin pérdida de paquetes (Mellanox-Kernel) |
| Custom Micro-Mainframe | Compuesto térmico propio (CrimsonLayer TIM) — 11 elementos, punto crítico 400°C |

## Proyectos documentados

- **Mainframe eGPU** — infraestructura CUDA/NAS portátil con Tesla V100 vía Thunderbolt
- **RoboNexis N1** — robot autónomo con IA embebida, LoRa Mesh y energía solar
- **NexCore** — supervisión jerárquica y validación de inferencia bajo saturación extrema
- **TouchPi Node** — nodo híbrido Edge AI (Raspberry Pi 4B + LattePanda)
- **Asus E210M** — optimización estructural de Windows 11 LTSC en hardware de entrada
- **Custom Micro-Mainframe** — chasis eGPU vía M.2 con compuesto térmico propietario
- **CryoStorm** — ingeniería térmica adaptativa sobre Acer Nitro 5
- **Hydra Prometheus X2** — refrigeración híbrida avanzada para RTX 3090

Cada proyecto tiene su propia página de detalle con métricas reales, tablas de resultados y stack tecnológico.
